# 3design-ui

## Install

```
yarn add @3-shake/3design-ui

# install dependencies
yarn add @mui/lab @mui/material @emotion/react @emotion/styled react-icons
```


## リリース方法

### 1. リリースバージョンの確認

https://github.com/3-shake/3design-ui/releases

上記のURLから、次のリリースバージョンを確認

<img width="1122" alt="CleanShot 2022-08-03 at 17 56 19@2x" src="https://user-images.githubusercontent.com/18649881/182567891-fbadc50e-f578-4305-a077-a6cc4c223989.png">


### 2. package.jsonのバージョンアップ PR を提出 / マージ

**example**
```
git checkout -b v2.1.5 origin/v2
```

**マージする際、prefixにreleaseをつけること**


### 3. ドラフトのリリースを、パブリッシュに変更

https://github.com/3-shake/3design-ui/releases
